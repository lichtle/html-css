Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>, obsoleta e não semântica.

Nesta frase temos um termo em destaque usando a tag <strong> semântica e atual.

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i>, obsoleta e não semântica.

Nesta frase temos um termo em ênfase usando a tag <em> semântica e atual.

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>. Ela é pouco usual mas pode ser utilizada.

Outro parágrafo com texto marcado para demonstrar o funcionamento de seletores CSS.

Textos grandes e pequenos

Estamos criando um texto grande (utilizando a tag obsoleta <big>) e um texto pequeno (utilizando a tag semântica <small>) nesse parágrafo.

A tag <small> acrescenta semântica ao texto pois é utilizada em contratos, por exemplo. Já a tag <big> se tornou obsoleta pois podemos apenas alterar seu tamanho com CSS, ela não acrescenta significado algum ao texto.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado. Para isso utilizamos a tag <del>

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Para isso utilizamos a tag <ins> Exemplo: Minha cor favorita é rosa lilás!

Existe também o sublinhado com a tag <u>, mas ela não é semântica.

Texto Sobrescrito

Para inserir coisas como x20+3 utlizamos a tag <sup>.

Texto Subscrito

Para inserir coisas como H2O utilizamos a tag <sub>.
