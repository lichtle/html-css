Código-fonte (partes de código) / Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

Usamos a tag <code> para monoespaçar a fonte (deixar todos os caracteres ocupando o mesmo espaço). Vejamos o próximo exemplo:

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print (f'O número {num} é PAR')
    else:
        print (f'O número {num} é ÍMPAR')
    print('Fim do programa')

Acima utilizamos a tag <pre> envelopando a tag <code> para manter o texto pré-formatado, exatamente da mesma maneira na qual ele foi digitado, incluindo quebras de linhas, espaços e tabulações.

Caso a tag <pre> não seja utilizada, o código aparecerá em formato de frase, e a linha só será quebrada ao antigir o limite da tela. Isso faz com que o código possa ser visualizado da maneira que está escrito no editor.
