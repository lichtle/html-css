Listas de definição

As listas de definição são aplicadas utilizando a tag <dl> (definition list). Esta deve englobar as tags <dt> (termo) e <dd> (definição). É como se fosse um dicionário com itens e suas definições. Veja o exemplo abaixo (tenha em mente que o pontilhado embaixo de HTML e CSS foram adicionados por padrão através da tag <abbr> vista no exercício ex008b):

HTML
Linguagem de marcação utilizada para a criação do conteúdo de um site.
CSS
Linguagem de marcação utilizada para a criação do design (aparência) de um site.
JavaScript
Linguagem de programação utilizada para a criação de interatividade de um site.
